# VanillaTodo List
VanillaTodo List is a simple and lightweight todo list application built using HTML, CSS, and vanilla JavaScript. It uses local storage to store user data, making it easy to access and use on any device. It is also responsive and can be used on any screen size.

## Getting Started
To get started with VanillaTodo List, simply open the `index.html` file in your web browser. The application will automatically load any previously saved todo items from local storage.

## Adding Tasks
To add a new task to your todo list, simply enter a task name in the input field and press the "Add Task" button. The new task will be added to the bottom of your todo list.

## Completing Tasks
To mark a task as completed, simply click on the task name. The task will be crossed out to indicate that it has been completed.

## Deleting Tasks
To delete a task, click on the "Delete Task" button that appears to the right of the task name. The task will be removed from your todo list.

## Clearing the List
To clear your entire todo list, simply click on the "Remove All" button. This will remove all tasks from your list.
